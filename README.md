# API RESTful Viajes Doñana - Ayuntamiento de Almonte

API RESTful desarrollada para el **Ayuntamiento de Almonte** como parte del proyecto **Viajes Doñana**. La API actúa como capa de conexión entre la **base de datos** y la **aplicación de escritorio**, permitiendo la gestión y el registro de **viajes y viajeros** de forma centralizada.

## Características

- API para la gestión de **viajes y viajeros**.  
- Conexión entre la **base de datos** y la aplicación de escritorio **Viajes Doñana**.  
- Registro, consulta y actualización de información de viajes.  
- Gestión de datos de viajeros asociados a cada viaje.  
- Diseñada para integrarse con una **aplicación de escritorio**.  
- Optimizada para uso interno del **Ayuntamiento de Almonte**.
